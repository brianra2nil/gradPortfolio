Coview19

Community driven review site for compliance with covid 19 regulations.

Users can view and enter reviews for businesses and rate them based on their adherance to mask use, social distancing and sanitation with a 5 star rating system. Collaborators: Matthew Javier, Danielle Hillman, Pat Downer. Group Project

Using MySQL database to store users ratings and comments. Use of RateYo NMP package for the star effect. Full Stack application with RESTful API

Google Books Search

A google books search using API

Users can search for a book and also make a save list. Each result comes with a link to the google page.

This is a full stack MERN app using MongoDB, Express.JS, React.JS and Node.JS.
